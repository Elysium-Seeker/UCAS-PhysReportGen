# 大物实验报告生成器 (PhysReportGen) v2.0

这是一个基于 Flask 和 AI 的物理实验报告自动生成工具，能够根据实验指导书（PDF）和实验原始数据（照片）自动完成报告撰写、数据处理和 Python 自动绘图。

## 核心功能
- **PDF 文本提取**：自动读取实验指导书中的原理、步骤等信息。
- **AI 报告撰写**：基于 GPT-4o 自动生成 LaTeX 格式的实验报告。
- **Python 自动绘图**：自动从报告数据中提取数值，生成符合物理实验要求的拟合图表，并自动嵌入 PDF。
- **一表一图排版**：智能识别报告中的表格，并将对应的分析图表交替插入。
- **LaTeX 实时编译**：一键编译生成完美的 PDF 实验报告。

## 安装说明
1. **安装 Python 依赖**：
   ```bash
   pip install -r requirements.txt
   ```
2. **安装 LaTeX 环境**：
   确保您的系统中安装了 `TeX Live` 或 `MiKTeX`，并确保 `xelatex` 在系统 PATH 中。
3. **字体要求**：
   模板默认使用 `SimHei` (黑体) 和 `SimSun` (宋体)。如果系统中没有这些字体，请将对应的 `.ttf` 文件放入 `latex_template/fonts` 目录。

## 使用步骤
1. 运行应用：`python app.py`。
2. 访问 `http://127.0.0.1:5000`。
3. 在“API 设置”中配置您的 OpenAI 或兼容接口（如 DeepSeek, OhMyGPT 等）。
4. 上传实验指导书（PDF）和原始数据记录表（照片）。
5. 确认学生信息，点击“开始生成”。

## 项目结构
- `app.py`: 应用主逻辑。
- `config.py`: 配置文件（可在其中修改学生默认信息）。
- `utils/`: 核心功能模块（AI后端、绘图执行、提取器等）。
- `latex_template/`: LaTeX 模板文件。
- `static/`, `templates/`: 前端界面。
